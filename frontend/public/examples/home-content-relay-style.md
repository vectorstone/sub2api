# One Key. All AI.

> 统一接入 Claude、GPT、Gemini 等主流模型。  
> 一个 API Key，完成多模型路由、账号池切换、透明计费与稳定交付。

---

## 为什么选择我们

- **统一网关**
  - 使用一个兼容接口接入多个 AI 服务
  - 降低接入成本，减少多平台切换

- **多账号池调度**
  - 自动在多个上游账号之间分配请求
  - 降低单账号限流、异常和中断带来的影响

- **透明计费**
  - 支持按量计费、配额限制、订阅管理
  - 每个用户、每个密钥、每个模型的使用情况都可追踪

- **稳定可控**
  - 支持分组、路由、优先级、故障切换
  - 适合个人、团队和内部 AI 平台统一接入

---

## 核心能力

### 统一模型入口
无需分别维护 Claude、GPT、Gemini 的不同接入方式。  
通过统一 API 入口，快速完成模型调用与业务集成。

### 高可用请求转发
通过账号池和策略调度，自动处理部分上游不可用、速率限制、临时故障等情况。

### 用量与成本可视化
查看调用量、Token 消耗、费用明细、余额和配额，帮助你更好地控制成本。

```bash
    "messages": [

        "role": "user",

      }

  }'
```
